A map-driven application for tree inventory, inspections, and work orders—built with ASP.NET Core MVC.

.NET 8 / ASP.NET Core MVC (C#) JavaScript Razor ViewModels/POCOs Bootstrap 5 Bootstrap Icons
Key Features
  • Tree inventory CRUD (species, DBH, condition, location, photos)
  • Inspection forms with risk rating and maintenance recommendations
  • Map filters by species, condition, status; clustering & search
Architecture
  • MVC controllers with ViewModels for inventory & inspections
