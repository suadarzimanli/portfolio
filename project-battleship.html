Battleship Assistant

Screenshots & short notes

← Back to Projects GitHub

A C# Windows Forms Battleship game with registration & login, a computer bot opponent, and SQL Server–backed user storage.

C# Windows Forms SQL Server ADO.NET (Microsoft.Data.SqlClient) Login & Registration Game Logic / RNG
Key Features
  • User registration & login backed by SQL Server
  • Play Battleship vs a computer-controlled bot
  • Randomized bot ship placement
  • Randomized bot shots with turn-based flow
Architecture
  • Windows Forms UI (boards, status panel, messages)
  • ADO.NET commands/parameters for data access
  • Simple repository layer for users/auth
  • Serializable models for game/grid state
