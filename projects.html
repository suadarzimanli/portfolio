Forecast Horizon
Crisis Zones Forecast Dashboard

GIS-GEOINT web app that maps historical ACLED events for Poland and surfaces 3-month forecasts via an interactive map, filters, a forecast panel, and live charts. Built with ASP.NET Core MVC and Google Maps.

ASP.NET Core MVC C# Google Maps API Chart.js GeoJSON ArcGIS Pro REST APIs Postman JSON
Key Features
  • Conflict map: loads ACLED Poland data (GeoJSON), clusters markers, and shows InfoWindows (event/sub-event, date, city, notes).
  • Forecast panel: reads poland_enhanced_forecasts.json, highlighting expected events/fatalities, likely protest types, and regional breakdowns.
  • Interactive filters: event type, sub-event type, city, year — updates map and charts in real time.
  • Data insights: Chart.js bar (events by type), line (fatalities over time), pie (events by city).
