Ticket Application Supportdesk

Screenshots & short notes

← Back to Projects GitHub

An ASP.NET MVC helpdesk that lets teams create, track, filter, and resolve tickets—with a dashboard, comments, and CSV export.

ASP.NET Core MVC Razor & Tag Helpers ViewComponents (QuickStats) ADO.NET (Microsoft.Data.SqlClient) Dependency Injection Configuration (appsettings) Data Annotations Anti-forgery CSV Export SQL Server
Key Features
  • Ticket management (CRUD): title, description, status, priority, category, assignee, customer email, due date, tags, resolution, timestamps
  • Comments: per-ticket threads, public vs internal notes
  • Dashboard: Open / In-Progress / Resolved / Urgent + Recent tickets
  • Lists: server-side search, filters, sorting, pagination
  • CSV export for current filtered/sorted view
Architecture
  • ASP.NET Core MVC with ViewModels
  • ViewComponent for QuickStats panel
  • ADO.NET repository (no EF) via ITicketRepository and DI
  • Config via appsettings.json; validation via DataAnnotations
  • Anti-forgery tokens on forms
